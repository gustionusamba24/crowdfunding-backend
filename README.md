
# Crowdfunding

## Description:

Crowdfunding is a way of raising money to finance projects or business. This project was inspired by [KitaBisa](https://kitabisa.com/). Feel free to try this API.

## Tech Stack:

* Go
* Gin
* GORM
* Validator
* MySQL
* JWT-Authorization
* 3rd Party APIs (Midtrans)
* REST
